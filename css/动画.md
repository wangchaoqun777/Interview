动画事件发生的过程：  
  - enter
    v-enter opacity: 0
    v-enter-to opacity 1

  - leave
    v-leave opacity: 1s 
    v-leave-to opacity: 0

1) 将需要添加动画的元素 添加到<transtion name=""></transtion>, name用来添加前缀
2） 在style里添加相关样式
    .v-enter,
    .v-leave-to: {
      opacity: 0,
      transfrom: translateX(150px)
    }

    .v-enter-to,
    .v-leave:{
      transtion: all .8s ease
    }
